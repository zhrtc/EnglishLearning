二、形容词/副词辨析

📌 a few/few / a little/little
可数不可数
肯定（有一些）a few 几个a little 一点
否定（几乎没有）few 很少little 很少
I have a few friends. / He has few friends. 我有几个朋友。/他几乎没有朋友。
📌 other / another / the other / others
another + 单数(另一) / other + 复数(其他的)
the other (两者中)另一个 / others = other people/things
the others = the other + 复数(剩余全部)
One is red, the other is blue. / Some like tea, others like coffee.
📌 each / every
each 强调个体（两者或以上） / every 强调整体（三者或以上）
each 可单独作代词 / every 只能作形容词
Each student has a book. / Every student likes sports. / Each of them is happy.
📌 some / any
some 用于肯定句 / any 用于否定/疑问句
但表建议/请求的疑问句用 some：Would you like some tea?
any 也可用于肯定句表"任何"：Any student can answer.
📌 many / much / a lot of
many + 可数复数 / much + 不可数
a lot of / lots of + 可数/不可数（用于肯定句）
否定句中常用 many/much 代替 a lot of
📌 every day / everyday
every day (分开) 每天(副词短语)
everyday (合起来) 日常的(形容词)
I go to school every day. / This is my everyday life.
📌 sometime / some time / sometimes
sometime 某时(副词) / some time 一段时间
sometimes 有时(频率) / some times 几次
I'll call you sometime. / I need some time. / I sometimes go there.
📌 altogether / all together
altogether 完全、总共 / all together 一起、全体
Altogether 50 students. / We sang all together.

三、其他常考易混词

📌 maybe / may be
maybe 副词(也许) = perhaps：Maybe he is right.
may be 情态+be(可能是)：He may be right.
📌 hard / hardly
hard 努力地/难的：He works hard. / It's a hard problem.
hardly 几乎不(表否定)：I can hardly believe it.
📌 too much / much too
too much + 不可数名词：too much water (太多水)
much too + 形容词/副词：much too expensive (太贵了)
📌 ago / before
ago + 时间段，用于一般过去时：I met him 3 days ago.
before + 时间点，用于完成时：I have seen it before.
📌 also / too / either / as well
also 放句中 / too 放句末(肯定句)
either 放句末(否定句) / as well 放句末(肯定句)
I also like it. / I like it, too. / I don't like it, either. / I like it as well.
📌 so / such
so + adj./adv. 或 so + adj. + a/an + n.
such + a/an + adj. + n. 或 such + adj. + 复数/不可数n
So beautiful a flower = Such a beautiful flower
So many people / Such good news
📌 among / between
among 三者或以上之间 / between 两者之间
She is among the crowd. / Between you and me.
📌 in / after (时间)
in + 时间段，用于将来时：He'll return in 2 days.
after + 时间段，用于过去时：He returned after 2 days.
after + 时间点，可用于将来：after 5 o'clock
📌 by / until
by 截止到...时(强调完成)：I'll finish it by Friday.
until 直到...时(强调持续)：I'll wait until Friday.
📌 because / because of
because + 从句 / because of + 名词
He didn't come because he was ill. / ... because of his illness.
