👥 主谓一致

语法一致 · 意义一致 · 就近原则 · either/neither/everyone等

📝 单选常考✏️ 写作基础

一、语法一致 主语决定谓语

📌 单数主语
each, every, either, neither, one 作主语→单数
Everyone is ready. / Each student has a book.
不定式/动名词/从句作主语→单数
Swimming is fun. / To learn English is important.
📌 复数主语
both, many, several, few + 复数 → 复数
Both of them are happy.
the + adj 表一类人 → 复数
The poor need help. / The rich should help.

二、就近原则 🔥 必考

📌 就近原则
either...or / neither...nor / not only...but also / there be → 谓语与最近的主语一致
Either you or he is wrong. 要么你要么他错了。
Not only you but also she knows the truth. 不仅你，她也知道真相。
📌 特殊情况
and 连接的两个名词指同一个人/物时→单数
The writer and poet is coming.（同一个人）
either/neither of + 复数 → 单数（正式）/ 复数（口语）
a number of + 复数 → 复数 / the number of + 复数 → 单数
A number of students are playing. / The number of students is 50.

三、意义一致

📌 集体名词
family / team / class / group / audience 视为整体→单数；考虑成员→复数
My family is large. / My family are all early risers.
📌 形复意单
news / maths / physics / politics → 以s结尾但为单数
The news is good. / Mathematics is my favorite.
trousers / glasses / scissors → 复数（需a pair of的配合）
