二、各代词用法详解

📌 人称代词（主格/宾格）
主格作主语：I love you. / He is a teacher.
宾格作宾语：Please help me. / I gave him a book.
排序：单数(you→he→I)，复数(we→you→they)
You, he and I are good friends. 你、他和我都是好朋友。
⚠ be 动词后用主格（正式）或宾格（口语）：It was I (formal) / It was me (casual).
📌 物主代词（形物代/名物代）
形物代 + 名词：my book / your pen
名物代 = 形物代 + 名词：Mine = my book
This is my bag. That is yours. 这是我的包，那是你的。
⚠ its（形物代）vs it's（it is缩写），注意区分！
📌 反身代词
主语和宾语是同一人时使用
I taught myself English. 我自学英语。
She hurt herself when she fell. 她摔倒时伤了自己。
固定搭配：by oneself (独自) / enjoy oneself (玩得开心)
I did it by myself. / They enjoyed themselves.
📌 指示代词 this/that/these/those
this 这个(近) / that 那个(远) / these 这些 / those 那些
This is my book. That is yours. 这是我的书。那是你的。
These are apples. Those are oranges. 这些是苹果。那些是橙子。
电话用语：I'm this. / Who's that?

三、不定代词 🔥 高频易错

📌 some / any
some 肯定句：I have some books.
any 否定/疑问句：I don't have any. / Do you have any?
但表示请求/建议的疑问句用 some：Would you like some tea?
📌 many / much / a lot of
many + 可数复数：many students
much + 不可数：much water
a lot of / lots of + 可数/不可数：a lot of people / money
📌 (a) few / (a) little
可数不可数
肯定（有一些）a few (一些)a little (一点)
否定（几乎没有）few (很少)little (很少)
I have a few friends. / He has few friends. 我有一些朋友。/他几乎没有朋友。
📌 other / another / the other
another + 单数(另一个)：I want another cup of tea.
other + 复数(其他的)：other students
the other 两者中另一个：One is red, the other is blue.
others = other + 名词(泛指其他人/物)
📌 both / either / neither
both 两者都(复数)：Both of them are students.
either 两者任一(单数)：Either answer is correct.
neither 两者都不(单数)：Neither answer is correct.
Both are good. / Either is OK. / Neither is right.
📌 all / none / each / every
all 全部(三者+ / 不可数)
none 全不(三者+) = not any
each 每一个(侧重个体) + 单数
every 每一个(侧重全体) + 单数
All students are here. / None of them knows.
Each student has a book. / Every student likes sports.
📌 复合不定代词
some/any/no/every + body/one/thing
人 (-body/-one)物 (-thing)
肯定somebody/someonesomething
否定/疑问anybody/anyoneanything
否定nobody/no onenothing
全体everybody/everyoneeverything
形容词修饰不定代词要放在后面：something new / someone important

四、it 的特殊用法 🔥 高频考点

📌 形式主语/宾语
形式主语：It + is + adj. + (for sb) + to do
It is important to learn English. 学英语很重要。
形式宾语：find/think/make + it + adj. + to do
I find it easy to swim. 我发现游泳很容易。
📌 强调句型
It is/was + 被强调部分 + that/who + 其余部分
It was Tom that broke the window. 是Tom打破了窗户。
It is English that I study. 我学的是英语。
📌 指时间/天气/距离
It is 8 o'clock. / It is raining. / It is 5 km.
